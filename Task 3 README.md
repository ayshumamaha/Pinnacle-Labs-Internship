# 📝 Personal Blog Management System

## 📌 Project Overview

The **Personal Blog Management System** is a Python-based console application that allows users to create, organize, manage, and maintain personal blog posts efficiently. The project uses **JSON** for data storage and provides an easy-to-use menu-driven interface for performing various blog management operations.

This application demonstrates practical implementation of Python concepts such as file handling, JSON manipulation, functions, exception handling, lists, dictionaries, loops, conditional statements, and modular programming.

---

## 🚀 Features

- Create new blog posts
- View all blog posts
- View complete blog details
- Search blogs by title
- Search blogs by author
- Search blogs by category
- Search blogs by tags
- Edit existing blog posts
- Delete blog posts
- Sort blogs alphabetically
- Display recently created blogs
- Find the longest blog
- Find the shortest blog
- View blog statistics
- Export blogs as HTML files
- Backup blog database
- Restore blog database
- Exception handling for invalid inputs

## ▶️ How to Run

1. Install Python 3.x.
2. Download the project files.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python blog_management_system.py
```

6. Use the menu to perform different blog management operations.
